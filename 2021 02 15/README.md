### 📒Contents

- 병합 정렬 `merge sort`
    + 배열을 앞부분과 뒷부분으로 나누어 각각 정렬한 다음 병합하는 작업을 반복해 정렬하는 알고리즘

- 병합 정렬 알고리즘의 순서
```
배열의 요소 개수가 2개 이상인 경우

1. 배열의 앞부분을 병합 정렬로 정렬
2. 배열의 뒷부분을 병합 정렬로 정렬
3. 배열의 앞부분과 뒷부분을 병합
```

- `O(nlogn)`
    + 안정적인 정렬
